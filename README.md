# Portafolio de Edson Aguilar

Bienvenido a mi portafolio profesional.  
Este proyecto está creado en **Laravel Blade + Tailwind 4 + Alpine.js**, con soporte para **modo oscuro**, **i18n (ES/EN)**, diseño tipo mármol, animaciones y un enfoque profesional y minimalista.

## Tecnologías

-   Tailwind CSS 4
-   Alpine.js
-   Animaciones personalizadas
-   Traducciones ES/EN
-   Diseño elegante inspirado en mármol (sin imágenes)
-   Código optimizado para Lighthouse

-   Listado completo de proyectos reales

-   Botones de contacto (WhatsApp, Email)
